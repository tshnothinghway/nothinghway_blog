# sychronized 使用
 - 类实例化作为对象传入给Runnable接口
 - 用该对象的同一个属性（new Object）
 - 作为传入参数对代码块进行枷锁
 - 保证只能被单一线程访问，保证数据被单一线程修改，从而使得数据符合应用场景
 - 在同步代码块中使用wait（）。并等待其他线程notify（）；
